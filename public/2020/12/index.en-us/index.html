Pre-process interface

  • recipe
  • prep
  • step
    • step_corr() : 다른 설명변수들과 상관관계가 높은 변수들을 제거
    • step_center()
    • step_scale()
### 질문
#### - all_outcomes의 의미는 무엇인가?
#### step_center, step_scale는 왜 저렇게 쓰는가?
iris_recipe <- training(iris_split) %>% 
  recipe(Species ~.) %>% # 전처리 시행
  step_corr(all_predictors()) %>% 
  step_center(all_predictors(), -all_outcomes()) %>% 
  step_scale(all_predictors(), -all_outcomes()) %>% 
  prep() # 전처리 시행

Model with a different engine

  • stan engine을 활용해서 bayesian approach를 시도해보자
# install.packages("rstanarm")
# set the prior distribution

prior_dist <- rstanarm::student_t(df = 1)
set.seed(123)

# make the parsnip model

bayes_mod <-
  linear_reg() %>% 
  set_engine("stan",
             prior_intercept = prior_dist,
             prior = prior_dist)

# train the model
bayes_fit <-
  bayes_mod %>% 
  fit(width ~ initial_volume * food_regime, data = urchins)

print(bayes_fit, digits = 5)
