list 중복 없애기

  • 참조 : https://bluese05.tistory.com/13

파이썬 리스트에서 중복을 없애기 위해서 이것저것 방법이 있겠지만 ‘set’ 자료형을 사용하면 쉽게 바꿀 수 있다.

  • ‘set’ 자료형의 특징은 아래와 같다.
    • 중복을 허용하지 않는다
    • 원소간 순서가 없다.

예제

a = [1,2,1,1,1,1,2,2,2,2,2,2,3,3,3,3]
print(a)
## [1, 2, 1, 1, 1, 1, 2, 2, 2, 2, 2, 2, 3, 3, 3, 3]
a = list(set(a))
print(a)
## [1, 2, 3]
