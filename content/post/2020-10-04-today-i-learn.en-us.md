### GitHub 저장소 이름 및 url 변경하기

1. github repository name 변경

2. git bash 해당 로컬 경로에서 프로젝트 저장소 및 url 출력

```
git remote -v
```

3. git remote set-url 명령어로 저장소 url 변경

```
git remote set-url origin "git url 작성"
```
