1. sep 옵션 in python

파이썬에서 여러 개를 출력할 때 sep 옵션을 걸 수 있음

print(1920, 1080, sep = 'x')
## 1920x1080
print('Hello', 'Python', sep = '-')
## Hello-Python

리스트 원소 합치기

  • 출처 : http://mwultong.blogspot.com/2006/12/python-list-merge.html
PA = ["pine", "apple"]

## 그냥 합치기
value = "".join(PA)
print(value)

## 구분자를 넣어 합치기
## pineapple
value2 = ", ".join(PA)
print(value2)
## pine, apple

진법 변환

  • 출처 : https://www.daleseo.com/python-int-bases/
## 10진법 -> 다른 진법
format(50, 'b') # 이진법
## '110010'
format(50, 'o') # 8진법
## '62'
format(50, 'x') # 16진법 (소문자)
## '32'
format(50, 'X') # 16진법 (대문자)
## '32'
format(50, 'X') # 16진법 (대문자)
## '32'
## 다른 진법 -> 10진법
format(int('2A', 16), 'd')
## '42'
format(int('62', 8), 'd')
## '50'
### 다른 진법 -> 다른 진법
format(int('2A', 16), 'o')
## '52'
format(int('62', 8), 'X')
## '32'

아스키 코드 <-> 문자열

print(ord('B')) # 문자를 아스키 코드로 변환
## 66
print(chr(102)) # 아스키코드를 문자로 변환
## f

리스트의 문자열을 int 형태로 변환

a = ['13', '4', '68']

a1 = list(map(int, a)) # 맵핑을 이용한 방법
print(a1)
## [13, 4, 68]
a2 = [int(i) for i in a] # 리스트 컴프리헨션을 이용한 방법
print(a2)
## [13, 4, 68]
